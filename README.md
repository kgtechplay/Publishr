# Publishr

Publishr is a modular, workflow-driven content production and distribution platform.

## Included scaffold
- Multi-service architecture under `services/` (gateway API, orchestrator, research/draft/generate/publish workers).
- Shared packages under `packages/` (`common`, `workflow_contracts`).
- Infrastructure/developer assets under `infra/`, `scripts/`, and `tests/`.
